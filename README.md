# Data Analytics Project

## Overview

This project demonstrates a complete **Data Analytics workflow** using Python, PostgreSQL, and Power BI.
The objective of the project is to collect, clean, analyze, and visualize data to generate meaningful business insights and support data-driven decision-making.

The project includes:

* Loading and processing datasets using Python
* Performing Exploratory Data Analysis (EDA)
* Cleaning and transforming data
* Running SQL queries on PostgreSQL
* Building interactive Power BI dashboards
* Creating analytical reports and insights

## How to Run the Project

### Prerequisites

Install the following before running the project:

* Python 3.x
* PostgreSQL
* Power BI Desktop
* Jupyter Notebook

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/data-analytics-project.git
```

Navigate to the project folder:

```bash
cd data-analytics-project
```

Install required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### Run the Project

1. Open Jupyter Notebook
2. Run the Python scripts/notebooks
3. Connect PostgreSQL database
4. Execute SQL queries
5. Open Power BI dashboard file (`.pbix`)
6. View reports and insights
